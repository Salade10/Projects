# PROWLER

Prowler is a CLI tool for word list generation It generates combinations of words based on what you give it. 
This is useful in ethical hacking for generating password lists.

The Commands include:

- -w The base word can be word or file
- -o file to write output to
- -p adds a prefix 
- -s adds a suffix
- -l enables leetspeak substitutions using the predefined dictionary
- -ls overrides default leetspeak map
- -irp includes prefixes from files
- -irs includes suffixes from files
- -cbn generates all combinations of input
- -cap turns of capitalisation
- -i starts the program
