# SIMPLE IMMUNE SYSTEM SIMULATION

This simulation shows a basic immune response to bacteria in the body. It produces a graphed output using matplotlib. 
The immune cells use a simplified version of cytokines to navigate towards the pathogen.

Constants:

- max_stride
- agent_speed
- spawn_distance
- pathogen_replication chance
- kill_interval
- killer_mutation_chance
- agent_detection_radius
- cytokine_decay_rate
- cytokine_count
- cytokine_response_cooldown
- max_cytokines
- max_agents
- step_size
- start_pathogen_amount
- cytokine_recruitment_threshold
